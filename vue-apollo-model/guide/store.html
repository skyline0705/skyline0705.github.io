# Store

store是用于全局存储Model实例的集合，后续会在Model中提供跨Model引用的模式。

# Multi Client

0.4.0版本以后，vue-apollo-model支持多clients的设置，例如

new Store({
    defaultClient,
    clients: {
        aClient,
        bClient,
    },
});

其中defaultClient为必传项，clients为选填项，在具体的apolloQuery或apolloMutation中，开发者可以自行定义所使用的client实例，当clients并不包含指定的key时，vue-apollo-model会默认回退使用defaultClient进行请求。
